### Deployment: https://aleksei-brodiagin.github.io/minesweeper/
***
### Clone repository:
```
git clone https://github.com/Aleksei-Brodiagin/minesweeper.git
```
### Install:
```
npm i
```
***
### Scripts:
1. Start server:
```
npm run start
```
2. Build with development mode:
```
npm run build:dev
```
3. Build with production mode:
```
npm run build
```
4. Watch:
```
npm run watch
```
5. Test:
```
npm run test
```
